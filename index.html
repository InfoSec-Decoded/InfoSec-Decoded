About

InfoSec Decoded brings together a range of expertise to educate, understand and address real-world challenges in cybersecurity and to equip the next generation of cybersecurity experts.

Basic and Low-level Python Network Attacks

This video course begins by showing, how to get Target and Attacker virtual machines running and properly networked.Then learn to connect to a server,automate login requests, write custom scanning tools and send unusual patterns of packets. Write your own tools to break into vulnerable websites.Finally use these tools to make defenses that distract attackers and detect suspicious network activity.
